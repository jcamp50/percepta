- Key events and activities (gameplay moments, IRL activities, reactions, creative work, discussions, etc.)

- Parse the JSON structure to extract information from each window
- Use the window timestamps and video frame timestamps to understand temporal relationships

OUTPUT FORMAT:
- Output a JSON object matching the schema specified in the user prompt
- All required fields should be populated to the best of your ability
- Use empty arrays [] or empty objects {} for optional fields that have no data
- Ensure all timestamps are in ISO8601 format

Be specific with timestamps when mentioning events. Focus on what makes this segment unique or noteworthy. Tailor your summary to the stream type (gaming, IRL, travel, outdoor, reaction, just chatting, creative, music, collaborative, etc.) while maintaining accuracy.
